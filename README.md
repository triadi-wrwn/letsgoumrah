# Lets Go Umrah web

This project was built with:

- [React 18](https://react.dev/)
- [Shadcn](https://ui.shadcn.com/) ( [Radix UI](https://www.radix-ui.com/) )
- [Tailwindcss](https://tailwindcss.com/)
- [pnpm](https://pnpm.io/)
- [Vite](https://vitejs.dev/)

## Clone and run

- `pnpm install`
- `pnpm run dev`
